# 파이썬 기본 문법 공부하기
### 01. 시작하기
- python3 설치
- Visual Studio

### 02. 데이터 타입
- '==' or '!=' 는 '값'을 비교할 때 사용
- 'is' or 'is not' 참조 비교, 객체 비교할 때 사용

### 03. 기본 자료 구조
- Container(list, set, dict, tuple, str): 여러 원소들을 가지고 있는 자료 구조
- Container 자료구조 모두 in 연산자로 멤버쉽 테스트 지원
- dict 순회시, key 목록만 순회 -> value 순회를 원한다면, .values() 함수 이용
